# Inventory Management System (CLI)

A command-line inventory management system developed in C with SQLite,
designed to simulate a real-world corporate application.

## Overview

This project focuses on building a modular and scalable CLI application
with persistent data storage, user authentication, and role-based access control.

## Features

- User authentication and authorization
- Inventory management with full CRUD operations
- Persistent storage using SQLite
- SQL-based reports
- Interactive command-line interface

## Technologies

- C
- SQLite
- CMake

## Project Status

Planning and initial setup phase.
